How to upload my HTML Code on internet

Today i am going to give you a step by step Tutorial on how
to upload your HTML code on Internet using GitHub Pase

The list Bellow are steps on how to upload HTML files. GitHub Pages is designed
to host your personal, organization, or project pages from a GitHub repository.

  1. Open GitHub webiste / github.com
  2. Create an account
  3. Your Coding shouild be on your local drive
  4. On GitHub page dashboard, Select Create new Repository(Folder)
  5. Rename the folder and select Public
  6. Select create respository
  7. On next screen select Upload an existing files
  8. Drag and drop files on pan provided / use upload option
  9. if all your files in a folder, open the folder and select all files , then drag and drop on the Pan
  10. Find all yoiur coding and select commit changes button

How to View Your Code on Internet ofter uploading them??

  • Select setting on GitHub , and select PAGES.
  • Select Main and save changes
  • Refresh the GitHub and you find your URL of all your files
  • This means that the ciode has been successfully published on internet
  • Right click on the link provided and select view on new window
  • You may encounter 404 Error Message,

How to solve 404 Error when open your website link??

When you try open your website link, a 404 error may appear , and here how you solve this error;;;;;

  • Go back to GitHub dashboard, Select Code . This will show all the code uploaded.
  • Rename the main file to inndex.html
  • Select commit Changes
  • Refresh your website and its Sorted!!!
